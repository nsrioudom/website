SFSU Virtual Nite

ABOUT

The current pandemic has brought distance between our San Francisco State University community. SFSU Virtual Night is an opportunity to connect with other students and meet new people through student hosted events such as movie night, game night, or anything you’d want to build a virtual event for. Students can post events they want to host and look for events to take part in.
